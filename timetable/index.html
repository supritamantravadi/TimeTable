How to define a table?
Using the table element in html
When to use tr
You use the tr element to start a new table row
When to use th
You use the th element after the tr element to indicate that the cells in the row are header cells
When to use thead
You use the thead element to create cells (td or th) at the very top row of the table
When to use tfoot
You use the tfoot element to create cells at the very bottom row of the table
Caption
You use the caption element to create a title for the table above it
What is the difference between colspan and rowspan?
You use the colspan element to merge cells horizontally (across columns) and you use the rowspan element to merge cells vertically (across rows)
What is the difference between cells spacing and padding?
Cells spacing defines the space between cells, and padding defines the space between a cell's content and borders
How do you define table borders using CSS?
You can define table borders using the border-style and border-color functions in CSS
How do you control border widths using CSS?
You can control the border widths using the four functions border-left-width, border-right-width, border-top-width, and border-bottom-width in CSS
